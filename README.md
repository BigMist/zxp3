# zxp3

A Sinclair ZX Spectrum +3 FPGA implementation


### Implements a standard ZX Spectrum +3 computer and the follwing hardware:

- Turbosound
- Tape loading
- Two floppy disk drives
- ZXMMC disk images
- Joystick Kempston with tow fire buttons
- IF2 joysticks
- RGB PAL or VGA (50 Hz) video output.

Supports any +3 compatible ROM, like Amstrad's standard ROM and Garry Lancaster's +3e ROM.  
Default ROM file should be copied in SD root directory as "zxp3.rom", it can be changed in any time though OSD (F12).


### Keyboard shortcuts

- Spectrum Reset: F9 or Ctrl+Alt+Del
- FPGA Reset: F8 or Ctrl+Alt+Backspace
- Toggle RGB/VGA: scroll lock or keep pressed F12 one second
